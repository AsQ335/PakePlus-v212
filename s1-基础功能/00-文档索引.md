## 🎯 使用场景指南

### 场景 1：第一次集成

**步骤**：
1. 阅读 **集成总结.md** 了解项目
2. 按照 **快速开始.md** 进行集成
3. 使用 **快速开始.md** 中的验证清单测试

**预计时间**：15-30 分钟

### 场景 2：自定义配置

**步骤**：
1. 阅读 **配置说明.md** 了解所有配置项
2. 根据需求修改 `onechat-custom.js`
3. 重新构建和测试

**预计时间**：30-60 分钟

### 场景 3：添加新功能

**步骤**：
1. 阅读 **架构说明.md** 了解模块结构
2. 阅读 **配置说明.md** 中的"自定义修改规则"部分
3. 在 `PageModificationsModule` 中添加新功能
4. 测试和验证

**预计时间**：1-2 小时

### 场景 4：问题排查

**步骤**：
1. 查看 **快速开始.md** 的常见问题部分
2. 查看 **README.md** 的调试技巧部分
3. 使用浏览器开发者工具查看日志
4. 如果需要，查看 **集成对比说明.md** 的回滚方案

**预计时间**：15-30 分钟

### 场景 5：性能优化

**步骤**：
1. 阅读 **架构说明.md** 的性能优化策略
2. 阅读 **配置说明.md** 的性能优化配置
3. 根据实际情况调整配置
4. 监控和测试性能

**预计时间**：1-2 小时
